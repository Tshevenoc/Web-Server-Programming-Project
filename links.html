Links

     Links are a must for navigating between the pages of your website. To create a link an anchor tag (<a>) is placed around the word that will be the link. In order for a link to work, the target attribute must be present and have the url or local document. Another useful attribute that is used is target. Target specifies where the link is opened and can be '_black' which opens a new window, '_self' which opens the link in the same window, '_parent' which opens it in a parent frame, '_top' which opens it in the full body of the window, or framename which is the specified name of a frame on the window and will then have the opened document.
