Introduction

     EasyFunWeb aims to provide a simple way to learn HTMl in order for anyone to create their own websites. Without access to classes, learning HTML takes time when feedback on proper coding practices and the uses of different tags is not given. Each lesson will be taught with a game, similar to the one below, followed by a video tutorial.

Index

In this section of the site definitions of common terms that correspond with the topic discussed will be placed below.


Html
- (Hypertext Markup Language) The standard language used to create websites.
CSS
- (Cascading Style Sheet) Standard language used to create the layout of a webpage. Can be inline, or in a linked document.
Javascript
- A high-level programming language that provides functionality in websites.
