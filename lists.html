Lists

     Lists provide an way to display grouped information together in a clear and easy to understand way. The first tags used in HTML lists are either <ol> or <ul> which will denote whether the list will be ordered by numbers or letters, or have bullet points for each item. To create an item in the list <li> is used.
     Another form of the list is the description list (shown in the index) denoted by <dl>. The tag <dt> is the description term, while <dd> is the description of the term.
