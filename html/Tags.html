Tags

Tags are one of the most basic parts for an HTML webpage, and denote how the elements are formatted depending on their tag.

The first tags that would be used in a webpage would be the HTML tag, head, and body tag. These three tags set up the basis of each webpage. The head tags hold all linked documents, suach as style sheets and javascript, the title of the webpage, built in style sheets, and other such as these, The body tag contains the main content of the webpage. Other common tags are the header and footer tag, as well as paragraph <p> and div tag. The paragraph tag holds text while the div tag seperates a portion of the screen for itself.
